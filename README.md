# SafeSnap Website

Marketing website for [SafeSnap](https://github.com/safe-snap/safesnap-chrome-extension) - a privacy-first Chrome extension that automatically detects and protects PII in screenshots.

## Development

```bash
bun install
bun dev
```

## Build

```bash
bun run build
```

Static export outputs to `out/`.

## Stack

- Next.js 16
- Tailwind CSS 4
- TypeScript
